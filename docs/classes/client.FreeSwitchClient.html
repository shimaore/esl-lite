Class FreeSwitchClient

FreeSwitchClient is the function you use to create a new client. It will automatically reconnect.

const client = new FreeSwitchClient()
client.on('connect', (service) => {
  // ^^ the connect callback is not async

  (async function() {

    const res = await service.bgapi('sofia status', 1000)
    console.log('sofia status is', res)

   })().catch( console.error )

})

client.connect()

service is a FreeSwichResponse object
