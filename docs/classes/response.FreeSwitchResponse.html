Class FreeSwitchResponse

The client will provide these when it connects

const client = new FreeSwitchClient()

client.on('connect', (service:FreeSwitchResponse) : void => {
  …
}

This class implements low-level APIs for FreeSwich, plus a couple higher-level ones (bgapi, command_uuid) suitable for use in application code.

bgapi

  • bgapi(command, timeout): Promise<FreeSwitchTimeoutError | FreeSwitchClosedError | FreeSwitchNoReplyError | FreeSwitchAbortError | FreeSwitchEventData>

  • Send an API command in the background. Wraps it inside a Promise.

    bgapi will return an error if the job submission fails.

    It will not return an error if the job itself failed. You can check the response from the background job in the response field of the return value.

    This method is not expected to throw / return a rejected Promise.

    Parameters

    • command: string
    • timeout: number

  • event_json(events): SendResult

  • Request that the server adds the events to its filter, and provide them in JSON format.
