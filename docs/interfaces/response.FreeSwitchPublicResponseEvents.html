Interface FreeSwitchPublicResponseEvents

Type definitions for all FreeSwitch events.

Note: one must call the .event_json() method in order for events to start. In other words, calling service.on(event,…), service.once(event,…) will not work unless reception of the event was requested using event_json.

  service.event_json(['CHANNEL_CREATE'])

  service.on('CHANNEL_CREATE', function (msg) {
    // Handle the new call
  }
